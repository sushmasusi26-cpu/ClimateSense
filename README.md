# 🌿 ClimateSense

AI-Powered Microclimate Risk Prediction System for Rural Farmers

---

## 📌 Problem Statement
Farmers lack accurate village-level climate prediction systems. 
Generic weather apps fail to provide hyperlocal risk alerts.

---

## 💡 Solution
ClimateSense collects real-time environmental data using sensors 
and predicts climate risks using AI models.

The system alerts farmers about:
- 🌊 Flood Risk
- 🌵 Drought Risk
- 🌧 Heavy Rain
- 🌱 Crop Stress

---

## 🏗 System Architecture
Sensors → ESP32 → AI Model → GSM/WiFi → Farmer Alerts

---

## ⚙️ Tech Stack
- Embedded C (Arduino)
- Python (Machine Learning)
- ESP32 / Arduino
- DHT11 Sensor
- Soil Moisture Sensor
- GSM Module
